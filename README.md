# Dictionary-Pdf-Viewer
This is an application made using C# . The application purpose is to provide an embedded dictionary while viewing pdf's so that student or readers do not have to google meaning of words they find difficult. They can simply highlight and search for meaning of words within the application. 

--> Make sure you are using Visual Studio to open the application. 
--> The language of the application is C# , make sure you have basic knowledge of C# and GUI using C# to understand the code.
--> The application also make the use of JSON so make sure you know about the serialization and deserialization of the JSON objects using C#.
